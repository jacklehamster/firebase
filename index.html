This website contains demos built in JavaScript with Firebase

Documentation: https://github.com/jacklehamster/firebase/

  • Social Gamedev v.1


    This is a social game development experiment, where you can play and edit the game collaboratively in real time. More specifically, you can draw enemies and choose their location, and play the game all at the same time. The project is collaborative so everyone can interact with it at the same time.


    Source code: social-gamedev

  • Dynamic Image / Image Reader


    Produces a placeholder for an image shared by everyone that can be added on any website. It changes dynamically depending on whoever uploaded an image last. Image reader is a read only version of Dynamic Image

    Example for Image reader:
    To update image: click on an image, select a new image to upload
    Source code: dynamic-image
    Source code: image-reader

  • Firebase Cam


    Uses firebase and Flash to broadcast live video from your webcam


    Source code: firebase-cam

  • World Canvas


    An infinitely large shared whiteboard that updates in real time


    Feel free to draw in the canvas
    Source code: world-canvas

  • Translation table


    A table where we can add data. Used mainly for storing translations in Firebase database.


    Feel free to change the data above, those are just sample data for demo.
    Source code: data-table
    The table uses third-party code: Columns

    A dialog based game where players can add their own dialogs and come up with their own endings
